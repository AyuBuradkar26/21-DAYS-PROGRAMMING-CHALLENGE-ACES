HDD Vs SSD

HDD SSD
HDD stands for Hard Disk Drive SSD stands for Solid State Drive
Expensive, roughly $0.20 per gigabyte Very cheap, only around $0.03 per gigabyte
Typically not longer than 1TB for notebook size drives; 4TB max for desktops Typically around 500 GB and maximum for notebook size drives; 10TB max for desktops
It implements the basic constrained network topologies. This is used to restrict mobility movements In the case of IPv6, it provides the embedded interoperability and mobility capabilities for the networking devices
Around 10-13 seconds average bootup time Around 30-40 seconds average bootup time
